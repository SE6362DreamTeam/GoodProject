Web Application Instructions

Add a URL to the Database

1. Click on Add URL on the Home page.

2. Enter a term that you would like to enter into the database.

3. Enter the corresponding URL into the database.

4. Click the Submit button.

If your URL was accepted, it will appear as an appended entry in the database contents.

Scrape Web Pages

1. Click on Scrape Web Pages on the Home page.

2. Wait for the application to scrape contents from all URLs in the database.

If the operation was successful, a “Web Pages Scraped Successfully” message will appear.

Search

1. Click on Search on the Home page.

2. Enter your search term.

3. Select your search type: AND, OR, NOT.

4. Check the box if you would like a case sensitive search performed.

5. Click the Submit button.

Perform KWIC Operations

1. Click on Scrape Web Pages on the Home page.

2. Wait for the application to scrape contents from all URLs in the database.

If the operation was successful, a “Web Pages Scraped Successfully” message will appear.

3. Return to the Home page by clicking Home in the top left corner.

4. Click on Circular Shift Demo on the Home page.

5. Use the controls to pick how many results to show at a time:

  • Select the number of records to display per page using the dropdown menu.
  • Click the Previous or Next button to navigate through the pages of results.

6. All alphabetized, circular shifted lines of the descriptions of all URLs in the database should appear incrementally based on the selected settings.
